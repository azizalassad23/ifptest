01 VARIABLE & DATA
Variabel adalah "wadah" untuk menyimpan data. Di Python, kita tidak perlu mendeklarasikan tipe data secara manual.

Tipe data umum:
  • String: Teks (diapit kutip ' ' atau " ").
  • Integer: Bilangan bulat.
  • Float: Bilangan desimal.
nama = "Siswa Cerdas" # String
umur = 16 # Integer
tinggi = 170.5 # Float

print(nama)
02 LOGIC CONTROL (IF/ELSE)
Komputer mengambil keputusan menggunakan logika perbandingan.
PENTING: Python menggunakan Indentasi (spasi di awal baris) untuk menandai blok kode.
nilai = 80

if nilai >= 75:
    print("Lulus Kompeten")
else:
    print("Perlu Remedial")
03 LOOPS (PERULANGAN)
Jangan menulis kode yang sama berulang kali. Gunakan for loop untuk melakukan tugas otomatisasi.
Fungsi range(5) artinya mengulang sebanyak 5 kali (mulai dari 0 sampai 4).
# Mengulang 5 kali
for i in range(5):
    print("Saya suka Coding!")

# Output:
# Saya suka Coding! (x5)
